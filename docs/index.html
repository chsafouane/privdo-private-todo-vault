Features

Everything you need, nothing you don't

A focused task manager with real encryption. No bloat, no telemetry, no ads.

🔐

PIN-Protected Vault

All data encrypted with AES-256 before storage. Your PIN derives the key via PBKDF2 (600k iterations) and never leaves your device.

📋

Multiple Lists

Organize tasks into named lists. Create, rename, delete, and switch between lists with a single tap.

☁️

E2E Encrypted Sync

Optional cross-device sync. The server stores only opaque encrypted blobs — it can never read your data. Works via passphrase or email.

✋

Drag & Drop

Reorder tasks by dragging. On mobile, swipe left to delete with a 5-second undo window.

⏰

Deadlines & Alerts

Set due dates on tasks. Overdue items are highlighted in red. Desktop notifications when tasks are past due.

📴

Offline First

Works without internet. Your local vault is always the source of truth. Sync catches up when you're back online.

🔍

Search

Quickly filter tasks by text within the active list. Real-time results as you type.

🌙

Dark / Light Mode

Follows your system preference by default, or toggle manually. Both themes are carefully tuned.

💾

Export / Import

PIN-encrypted JSON backups. Import supports both vault and legacy formats for easy migration.

Security

Zero-knowledge by design

Your data is encrypted before it touches any storage. The sync server never sees plaintext — ever.

Local Cipher
AES-256 (CryptoJS)
Key Derivation
PBKDF2, 600k iterations
PIN Storage
Hash only — never plaintext
Sync Model
Zero-knowledge server
Data at Rest
Encrypted vault blob
Network Calls
Zero unless sync enabled
